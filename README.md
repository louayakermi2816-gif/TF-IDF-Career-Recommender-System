# Project 3: TF-IDF Career Recommender System

## 📌 Project Overview
This project represents the third milestone of an Industrial Training Kit. It demonstrates the fundamentals of **Information Retrieval and NLP**, where recommendations are generated by calculating mathematical similarity between text vectors instead of using pre-defined hardcoded rules.

## ⚙️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data Pipeline), Scikit-Learn (Vectorization & Math Engine)
* **Logic:** TF-IDF Vectorization & Cosine Similarity Scoring

## 🏗️ Architecture (Vector Space Model)
1. **Input & Sanitization:** Captures user technical skills, handles multi-word tokens (e.g., `machine_learning`), and normalizes casing to ensure a perfect match with the vocabulary.
2. **Vectorization (TF-IDF):** 
    * **Term Frequency (TF):** Measures how often a skill appears in a profile.
    * **Inverse Document Frequency (IDF):** Mathematically punishes generic skills while rewarding rare, highly specific skills.
    * Transforms both the user's profile and an external dataset of 54 job roles into dense numerical vectors sharing the exact same coordinate system.
3. **Scoring & Ranking:** Calculates the **Cosine Similarity** between the user vector and all job vectors to determine the mathematical overlap (measuring direction rather than magnitude). The engine sorts the scores descending and isolates the Top 3 optimal career paths.
